Lyra – Video Recommendation System

Lyra is a video recommendation system that suggests personalized videos to users based on their watch history. The project uses a deep learning–based collaborative filtering approach to learn user preferences and recommend relevant content.

The system includes a trained recommendation model, a backend API to serve predictions, and a simple frontend to display recommended videos.


Tech Stack

-Python

-TensorFlow / Keras

-FastAPI

-React

-CSV datasets


What This Project Shows

-Applied machine learning on real user–item interaction data

-Built and trained a neural recommendation model

-Exposed ML predictions through a backend API

-Integrated ML with a basic web application
